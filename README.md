## Usar como extensión

Este repositorio puede ser añadido como una **extensión** en MakeCode.

* abra [https://github.com/maximo2010-pixel/xd/releases](https://github.com/maximo2010-pixel/xd/releases)
* haga clic en **New Project**
* haga clic en **Extensiones** en el menú del engranaje
* buscar **https://github.com/maximo2010-pixel/xd/releases** e importar

## Edita este proyecto

Para editar este repositorio en MakeCode.

* abra [https://github.com/maximo2010-pixel/xd/releases](https://github.com/maximo2010-pixel/xd/releases)
* haga clic en **Import** luego haga clic en **Import URL**
* pegue **https://github.com/maximo2010-pixel/xd/releases** y haga clic en importar
